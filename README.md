# BashLab - Shell Scripting Exercises

This repository contains a collection of Bash scripts developed as part of laboratory exercises. The project focuses on practical applications of shell scripting, automation, and system administration tasks in a Linux environment.

## Project Scope
The scripts included in this repository cover various aspects of Bash programming, including:
* **File Management**: Automated creation, moving, and cleaning of directories and files.
* **Text Processing**: Using tools like `grep`, `awk`, and `sed` to parse and transform data.
* **System Automation**: Scripts designed to streamline repetitive terminal tasks.
* **Permissions & Security**: Examples of managing file access levels and script execution rights.
* **Process Control**: Simple logic for monitoring or managing system processes.

## Key Features
* **Automation**: Ready-to-use scripts for laboratory task requirements.
* **Logic Structures**: Implementation of loops (`for`, `while`), conditional statements (`if-else`), and functions.
* **Best Practices**: Focus on script readability and efficient use of standard Unix utilities.

## Requirements
To run the scripts in this repository, you need:
* A Linux-based operating system (or WSL on Windows).
* Bash shell (standard in most distributions).
